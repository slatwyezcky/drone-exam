# 🚁 Drone Pilot Exam Prep

Приложение для самостоятельной подготовки к экзамену по БПЛА.  
Включает **49 вопросов** по 4 темам:  
- Профессиональный цикл  
- Аэронавигация  
- Метеорология  
- Обзор приложения DJI  

📌 **Особенности**  
- Каждый вопрос сопровождается 4 вариантами (1 правильный + 3 правдоподобных дистрактора).  
- Сначала нужно выбрать ответ, затем нажать «Проверить» (можно изменить выбор).  
- Мгновенная обратная связь: зелёный — правильно, красный — ошибка.  
- После проверки кнопка меняется на «Далее».  
- В конце теста — итоговый результат и подробный разбор ошибок (ваш ответ → правильный ответ).  
- Вопросы каждый раз перемешиваются, порядок вариантов тоже случайный.

## 🚀 Быстрый старт (локально)

### 1. Клонируйте репозиторий
```bash
git clone <url-вашего-репозитория>
cd drone-exam-app
```

### 2. Установите зависимости
```bash
npm install
```

### 3. Запустите dev-сервер
```bash
npm run dev
```
Приложение будет доступно по адресу http://localhost:5173

## 📝 Замечания
- При первом запуске все вопросы отображаются в случайном порядке.
- Чтобы пройти тест заново, на странице результатов нажмите **«Пройти тест заново»** — вопросы снова перемешаются.
- Все данные хранятся локально, серверная часть не требуется.
